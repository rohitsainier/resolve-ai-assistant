Vision: a 5-year-old can edit in Resolve with words

We've covered the fundamentals — transcription, markers, rough cuts, shorts, subtitles, chapters, and single-shot prompt editing. Big creative tasks (effects, titles, color, audio, multi-cam) still need a human. Roadmap below.

✅ What works today (shipped)

🎤 Transcription done

  • Whisper local transcription of the active timeline
  • Word-level timestamps (enables filler detection)
  • Smart cache — invalidates on clip reorder / trim
  • Live progress bar with heartbeat

🟢 Smart markers done

  • 🟢 Highlights (key moments)
  • 🔴 Dead air (long pauses)
  • 🔵 Short-clip candidates (60–90s)
  • 🟡 Chapters (YouTube-style)
  • Preview & approve each before applying

🪶 Filler detection done

  • Finds every um / uh / like / you know
  • Word-level precision (±50ms padding)
  • Shows up as red markers or fed into rough cut

✂ Rough cut done

  • New timeline with dead-air regions removed
  • Rebuilds from original source media
  • Keep regions inverted from DEAD_AIR markers

✂ Shorts timeline done

  • New timeline containing just SHORT_CLIP regions
  • Multi-clip spans handled correctly
  • Concatenated in timeline order

📚 YouTube package done

  • Auto chapter markers on timeline
  • Description file (hook + chapters + hashtags)
  • .srt & .vtt subtitle export

💬 Prompt editing mvp

  • "Mark every time I mention AI"
  • "Find the 3 funniest moments"
  • "Clear all red markers"
  • Single-shot planner → executes actions

🗑 Marker management done

  • Clear ALL markers
  • Clear by specific color
  • 15 Resolve colors supported

🔄 Provider choice done

  • Anthropic Claude or OpenAI
  • Auto-detect by which key is set
  • AI_PROVIDER to force one
🚧 Coming soon (on the roadmap)

🤖 Iterative prompt mode next

  • Multi-turn tool-use (not single-shot)
  • LLM can search transcript, preview, refine
  • "Make it punchier" → iterative tightening

🗣 Speaker diarization planned

  • Who said what (whisperX / pyannote)
  • Per-speaker marker colors
  • Context-aware prompts ("find where Bob says...")

🎞 B-roll suggestions planned

  • LLM suggests insert points
  • Semantic search over media pool
  • Auto-place stock footage markers

🖼 Thumbnail picker planned

  • Extract frames at highlight markers
  • Vision model scores each for clickability
  • Export top-3 candidates

🎵 Beat-sync cutting planned

  • Detect music beats with librosa
  • Snap cuts to beat grid
  • Great for music video / montage edits

🎨 Color grading stretch

  • Suggest LUTs / node graphs
  • Analyze reference frames
  • Requires Resolve Studio API

🔊 Audio leveling stretch

  • Detect clipping / quiet patches
  • Suggest volume keyframes
  • EQ / noise reduction hints

🔒 Local-only mode planned

  • Ollama instead of cloud LLM
  • whisper.cpp instead of Python whisper
  • Fully offline editing for sensitive content

👤 Creator profiles planned

  • Per-creator style config
  • "Punchy cuts, max 8s shots, formal tone"
  • Fed into every prompt automatically

🎬 Title & caption generation planned

  • Auto-generate on-screen captions
  • Lower-thirds from transcript
  • Social-ready burned-in subtitles

📸 Multi-cam auto-switch stretch

  • Pick best angle per moment
  • Audio-level + face-detection heuristics
  • Auto build a synced multi-cam edit

🔁 Undo history planned

  • Log of all applied edits
  • Revert any past operation
  • Cross-session persistence
